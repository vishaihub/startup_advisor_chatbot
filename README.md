## 📌 Overview

A production-ready **Startup Advisor Chatbot** built using **Google Gemini API**.
It provides structured guidance on startup ideas, business models, and growth strategies with multi-turn conversation support.

---

## ⚙️ Tech Stack

* Python, Streamlit
* Google Gemini API
* AWS EC2
* python-dotenv

---

## 🏗️ Architecture

```
User → Streamlit UI → Backend → Prompt Module → Gemini API → Response → UI
```

---

## 🧠 Features

* Context-aware conversations
* Structured startup advice
* Secure API handling (.env)
* Modular code design
* Cloud deployment (EC2)

---

## ⚙️ Setup (Local)

git clone https://github.com/your-username/startup_advisor_chatbot.git
